Reddit Sentiment Stock Market Predictor

  • Tech Stack Transformers (Hugging Face), Python (Pandas, NumPy, Matplotlib, Seaborn, TensorFlow, scikit-learn, XGBoost), PRAW (Reddit API), yfinance, FinBERT, Jupyter Notebooks
  • Github URL: Project Link

Built a modular, end-to-end machine learning pipeline to predict stock prices by combining Reddit sentiment analysis with financial time series modeling. Fine-tuned FinBERT LLM (a BERT model trained on financial text) using labeled financial Twitter data, then applied the model to historical Reddit posts scraped from both general stock subreddits (e.g. r/stocks) and ticker-specific communities (e.g. r/AAPL). Generated daily sentiment scores from Reddit discussions and merged them with engineered features from Yahoo Finance data, including moving averages, lagged prices, returns, and volatility indicators.

Trained and evaluated multiple models — Random Forest, XGBoost, LSTM, and RNN — to forecast next-day stock prices. Visualized model predictions, feature importance, error metrics (MAE, RMSE, R²), and correlation matrices to compare performance across model types. Designed the system to be fully reusable: by modifying ticker symbol and subreddit list, the pipeline can predict prices for any stock with social presence and historical pricing data.
